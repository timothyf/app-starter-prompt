# app-starter-prompt

Generates prompts to help you get started building a web app.

Open `index.html` in a browser to use the app locally.

## GitHub Pages

This repository is configured to deploy to GitHub Pages via GitHub Actions on pushes to `main`.

After enabling Pages in the repository settings (Build and deployment source: **GitHub Actions**), the site will be available at:

`https://timothyf.github.io/app-starter-prompt/`
